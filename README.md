## webspeechdemo

Demo for web speech recognition. Requirements: Flask and icecream (install via pip) and xArm SDK:

```bash
  git clone https://github.com/xArm-Developer/xArm-Python-SDK.git && cd xArm-Python-SDK && python3 setup.py install
  ```
